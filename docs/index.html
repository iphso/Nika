About This Project

Nika uses neural implicit representations to encode videos as the weights of a neural network. Instead of storing pixel data directly, the network learns a continuous function that maps (x, y, t) coordinates to RGB values. This approach enables:

  • Compact representation - Video stored as network weights
  • Arbitrary resolution - Query any coordinate, not just pixel grid
  • Temporal interpolation - Generate frames between original timestamps

The visualizations show reconstruction quality metrics including PSNR (Peak Signal-to-Noise Ratio), spatial error heatmaps showing where the model struggles, and side-by-side comparisons with ground truth.
